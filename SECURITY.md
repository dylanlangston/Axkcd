# Security Policy

## Supported Versions

This project maintains security updates only for the versions listed below. If you are using an unsupported version, please upgrade to a supported version as soon as possible.

| Version | Supported          |
| ------- | ----------------- |
| >= 1.0   | :white_check_mark:|

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Email:** Send details to the maintainer at `mail@dylanlangston.com`.  
2. **Information to Include:** 
   - Description of the vulnerability  
   - Steps to reproduce  
   - Potential impact  
   - Suggested fix (optional)

3. **Response Time:** You can expect an initial response within 72 hours. The maintainer will work to acknowledge and address the issue as quickly as possible.

4. **Disclosure:** Once a vulnerability is confirmed and fixed, it will be publicly disclosed with a credit to the reporter, unless they request anonymity.

5. **Accepted/Declined Reports:** 
   - If accepted, the fix will be included in the next release.  
   - If declined, the maintainer will provide a brief explanation.

Your help in keeping this project secure is greatly appreciated.

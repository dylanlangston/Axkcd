# XKCDCore

XKCDCore: .NET library for XKCD and ExplainXKCD APIs. Type-safe, modern. Usable in any .NET application.

## Features

-   Type-safe classes for XKCD/ExplainXKCD APIs.
-   Async support for non-blocking I/O.
-   Fetches comics, explanations, metadata.
-   Integrates with dependency injection.

## Usage

Use `XKCDClient` and `ExplainXKCDClient` to interact with APIs.

## Development

### Prerequisites

-   .NET 10 SDK

### Getting Started

1.  Open solution.
2.  Restore dependencies:
    ```bash
    dotnet restore
    ```
3.  Build project:
    ```bash
    dotnet build
    ```

## Dependencies
